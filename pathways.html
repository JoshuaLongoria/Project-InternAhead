Cybersecurity Roadmap

Completed Months 1–3

Security Fundamentals

Build foundation in networking, systems, and security basics
  • Networking basics (TCP/IP, DNS, HTTP/HTTPS)
  • Security principles (CIA triad, authentication)
  • Linux system administration
  • Basic scripting (Python, Bash)
In Progress Months 4–6

Core Security Skills

  • OWASP Top 10
  • Cryptography basics
  • Firewalls & network defense
  • Nmap, Burp Suite
In Progress Months 7–9

Hands-on Practice & Certs

  • CTFs (HTB, THM)
  • Home lab
  • Security+ or CEH prep
  • Pentesting practice
Planned Months 10–12

Specialization & Career Prep

  • Pentesting / SOC / Cloud
  • Create portfolio
  • Apply to internships
  • Network professionally

Machine Learning Roadmap

Completed Months 1–3

ML Foundations

  • Python for ML (NumPy, Pandas)
  • Networking & security basics
  • Threat modeling for ML systems
  • Intro to ML (regression, classification)
In Progress Months 4–6

Core ML Security Concepts

  • Supervised learning (regression, classification)
  • Unsupervised learning (clustering, PCA)
  • Scikit-learn and model evaluation
  • Feature engineering and data preprocessing
Planned Months 7–9

Deep Learning and Projects

  • Neural networks with TensorFlow/PyTorch
  • RNNs/Transformers for NLP
  • CNNs for computer vision
  • Complete 2-3 end-to-end ML projects
Planned Months 10–12

Specialization & Portfolio

  • Specialize (CV, NLP, or MLOps)
  • Kaggle competitions and GitHub portfoliot
  • Deploy models (Flask, FastAPI, cloud platforms)
  • Apply to ML internships

Software Engineering Roadmap

Completed Months 1–3

Programming Fundamentals

  • Learn Python or JavaScript basics

Web Development Skills

  • Learn React and modern frontend

Interview Preperation

  • Mock interviews with peers
Planned Months 10–12

Application & Networking

  • Polish resume and LinkedIn profile
  • Attend career fairs and networking events
  • Apply to 50+ internship positions
  • Contribute to open source projects
